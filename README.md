# PHP 언어를 이용한 웹 바이브코딩 가이드 (2026.02.11)
- md 파일 작성시 마지막은 항상 개행문자로 끝남(종료표시)
## 세팅방법
1. ```.gitignore``` 파일과 ```_ukp``` 폴더를 복사 후 개발할 프로젝트에 붙여넣기.
2. ```.gitignore``` 파일 내 폴더 경로를 개발할 프로젝트에 맞게 변경.
3. 추가로 사용할 함수를 개발할 프로젝트의 ```class.ukp.php``` 파일 내에 추가하여 사용.
4. 사용할 폴더에 ```_.php``` 파일 생성 후 ```$global``` 변수값 세팅, ```$data``` 변수 빈배열로 초기화
## 신규 웹개발 하는경우
1. ```www/website``` 폴더 내용 복사, 웹 접근 가능한 폴더에 붙여넣은 후 ```_.php``` 파일 내용 수정.
2. 웹 접근 불가한 곳에 ```cli```, ```css```, ```js```, ```prompt```, ```view``` 폴더 추가.
3. ```prompt``` 폴더 내에 ```backend```, ```frontend```, ```ui``` 폴더 추가.
4. ```js``` 폴더에 ```ukp.js``` 파일 ```_.js``` 로 이름 변경하여 저장(자동완성을 위해)
5. ```_.php``` file 값에 불러올 파일명 입력 (예: ```_.php?file=<?php echo urlencode('index.css?v=1') ?>```)
6. 업로드를 위한 폴더 생성 후 프롬프트 작성시 해당 폴더에 있는 파일 업로드(작성속도를 위해)
## 규칙
- 프롬프트 파일은 .md 확장자로 작성
## 폴더구조 설명
- ```cli```: cli 환경에서 실행되는 파일. 웹접근 불가.
- ```css```: view 에서 페이지별로 불러오는 css 파일. 웹접근 불가.
- ```js```: view 에서 페이지별로 불러오는 js 파일. 웹접근 불가.
- ```prompt```: 소스생성을 위해 작성한 md 파일, 웹접근 불가.
- ```view```: 보여지는 페이지를 위해 사용되는 php 파일. 웹접근 불가.
- ```asset``` 은 정적파일용 폴더, ```upload``` 은 동적파일용 폴더.
## 파일구조 설명
- 웹루트에 접두어가 ```_``` 인 php 파일은 ajax 통신 전용 파일. ```$data``` 배열에 값 세팅 후 json으로 출력.
- ```view``` 폴더 php 파일에서는 웹루트 php 파일 변수중 ```$global```, ```$data``` 변수만 사용 가능.
- ```view``` 폴더내 접두어가 ```_``` 인 파일은 공통뷰를 의미, 아닌경우 페이지별 뷰.
- 파일명에 ```_``` 가 연속해서 두개이상 붙는경우 테스트용 파일이며 커밋되지 않는다.
## 프롬프트 작성 방법
- 기존 프롬프트 참고하여 작성
- 프롬프트 결과가 의도한것과 다르게 나올때 왜 그렇게 나왔는지 물어본 후 프롬프트 수정
- 하나의 명령을 여러줄에 걸쳐서 설명하면 이해도가 떨어짐
- 프롬프트 설명은 ```## 개별파일 상세설명``` 탭에 작성
## 개별파일 상세설명
- 파일 설명은 파일 이름 순서대로 작성
- 파일 이름 옆에 md 파일 수정일 입력, 각 md 파일 최상단 에도 수정일 입력

  - php: ```prompt/backend```
- 프롬프트 작성 시 필요한 ```prompt``` 폴더에 있는 ```upload_*.md``` 파일 업로드
- 생성된 소스코드 적용시 아래 참고하여 언어별로 주석 설정

  - css: ```/* #region @ukp {클래스명} {설명} {md 파일명} */```, ```/* #endregion */```
  - js: ```// #region @ukp {설명} {md 파일명}```, ```// #endregion```
  - php: ```// #region @ukp {설명} {md 파일명}```, ```// #endregion```
