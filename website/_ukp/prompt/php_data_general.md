# PHP 로 json 문자열을 출력하도록 작성해줘.
## 프롬프트
- 
## 출력값
- ```[code]``` 성공인경우 1, 실패인경우 2
- ```[msg]``` 1 인경우 성공, 2 인경우 실패
## 규칙
- 업로드한 ```ukp_function.md``` 파일에 나와있는 함수를 최대한 사용하여 작성해줘.
- ```$ukp->db_select_list()``` 함수 사용시 업로드한 ```db_select_list_query.md``` 파일에 select 쿼리 나와있어.
- ```$ukp->db_select_cnt()``` 함수 사용시 업로드한 ```db_select_cnt_query.md``` 파일에 select 쿼리 나와있어.
- php 5.2 버전 이상에서 동작하도록 작성해줘.
- 출력 내용은 ```## 출력값``` 섹션을 참고해주고 대괄호는 배열이라는 의미야.
