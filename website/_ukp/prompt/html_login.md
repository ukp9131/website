# HTML, CSS 코드 생성
## 프롬프트
- 아이디, 비밀번호를 입력할 수 있고, 아이디저장 체크박스가 있는 로그인페이지를 만들어줘.
- 아이디찾기, 회원가입 버튼은 필요 없어.
- form 태그 안에 제목 "" 로 설정해주고 서브타이틀은 필요없어.
## 설정값
- 최상단 클래스: ```ukpb__content```
## 규칙
- 아래 css 가 기본적으로 설정되어있다.
```css
* { margin: 0; padding: 0; box-sizing: border-box; font-family: "Noto Sans KR", sans-serif;}
```
- font-weight 속성은 thin, normal, bold 사용 가능하다.
- ```body``` 태그 내 html 요소를 생성하는 것이므로 body 태그로 시작하지 않는다. 
- 최상단 태그는 ```## 설정값``` 섹션에 ```최상단 클래스``` class를 가지고 시작한다.
- 상위요소 css 와 겹치는 것을 막기위해 생성된 css의 모든 선택자는 ``` .{최상단 클래스} > ``` 로 시작한다.
- 최상단 태그를 제외한 태그는 ```ukp``` 가 접두어인 클래스를 가질 수 없다.
- 모바일 레이아웃을 기본으로 한 반응형으로 작성한다.
