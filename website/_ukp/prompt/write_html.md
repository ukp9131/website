## 설정값
- 최상단 클래스: ```ukpb__content```
- 요소 클래스: ```{클래스명}```
## 규칙
- 아래 css 가 기본적으로 설정되어있다.
```css
* { margin: 0; padding: 0; box-sizing: border-box; font-family: "Noto Sans KR", sans-serif;}
```
- font-weight 속성은 thin, normal, bold 사용 가능하다.
- ```body``` 태그 내 html 요소를 생성하는 것이므로 body 태그로 시작하지 않는다. 
- 최상단 태그는 ```## 설정값``` 섹션에 ```최상단 클래스``` class를 가지고 시작한다.
- 상위요소 css 와 겹치는 것을 막기위해 생성된 css의 모든 선택자는 ``` .{최상단 클래스} > .{요소 클래스}``` 로 시작한다.
- 기존에 작성된 태그에 프롬프트 내용을 추가하는 것이므로 최상단 태그 css는 수정할 수 없다.
- 최상단 태그 내에 추가되는 요소는 ```## 설정값``` 섹션에 ```요소 클래스``` class를 가지고 시작한다.
- 최상단 태그를 제외한 태그는 ```ukp``` 가 접두어인 클래스를 가질 수 없다.
- 모바일 레이아웃을 기본으로 한 반응형으로 작성한다.
- html 파일과 css 파일을 분리해서 작성한다.
- HTML 태그 내에 PHP 코드(<?php ... ?>)를 절대 사용하지 않는다.
- 데이터가 출력되어야 하는 부분(값, 링크 등)에는 문맥에 맞는 임의의 **더미 데이터(Dummy Data)**나 # 링크를 넣어 완성된 화면처럼 보이게 작성한다.
- html, css 파일이 같이 업로드되는 경우 해당코드와 유사한 스타일로 페이지 생성해줘.
